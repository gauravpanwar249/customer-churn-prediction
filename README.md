# Customer Churn Prediction

A Machine Learning project that predicts whether a customer is likely to churn using customer information. The project uses XGBoost for prediction and Streamlit for an interactive web application.

---

##  Features

- Customer Churn Prediction
- Data Preprocessing Pipeline
- Exploratory Data Analysis (EDA)
- Random Forest Model
- XGBoost Model
- Cross Validation
- Hyperparameter Tuning (GridSearchCV)
- Streamlit Web Application
- Model Serialization using Joblib

---





##  Dataset

The dataset contains customer information such as:

- Gender
- Senior Citizen
- Partner
- Dependents
- Tenure
- Phone Service
- Internet Service
- Contract Type
- Monthly Charges
- Total Charges
- Payment Method

Target Variable:

**Churn**
- Yes
- No

##  Installation

Clone the repository


git clone https://github.com/gauravpanwar249/customer-churn-prediction.git



Install dependencies

```bash
pip install -r requirements.txt
```

Run the Streamlit app


streamlit run app.py

- Accuracy: **80%+**
